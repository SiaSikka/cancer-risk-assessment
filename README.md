# cancer-risk-assessment
A web based application for preliminary cancer risk assessment using HTML,CSS,JavaScript that simulates a basic cancer risk assessment based on user provided inputs.

## overview
This project aims to assess potential cancer risk based on user inputs and predefined parameters. It is intended for educational and research purposes only and not for medical diagnosis.

## Features
- User input-based risk evaluation
- Data processing and analysis
- Real time result display
- Clean and simple interface for interaction

## Tech Stack
- HTML
- CSS
- javascript
- Basic data handling concepts


## Note
This is a frontend-only project created for learning. This application is not a substitute for professional medical advice or diagnosis.
